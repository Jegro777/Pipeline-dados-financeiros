Arquitetura

O caminho de um dado, ponta a ponta

Uma API própria em FastAPI concentra a extração das cotações e expõe um único endpoint para o front-end. A evolução para cache, banco de dados e ETL está descrita no roadmap.

API Pública

CoinGecko e AwesomeAPI expõem endpoints REST com cotações atualizadas.

Coleta

O backend em FastAPI faz as requisições às APIs públicas em intervalos programados.

Tratamento

O backend valida e converte os dados antes de expor /api/cotacoes.

Atualização

O front-end consulta a API própria e atualiza a tela sem recarregar a página.

Dashboard

Os cards exibem valores e o horário da última sincronização.
